Чтобы посмотреть содержимое документации swagger-a не запуская приложение, нужно:
1. Скопировать содержимое файла api-docs.json
2. Перейти по ссылке https://editor.swagger.io/ и вставить скопированное

После запуска приложения доступна документация с отправкой запросов
ссылка: http://localhost:8080/swagger-ui.html
